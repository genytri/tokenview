# tokenview (Claude Code plugin)

This plugin wraps the [`tokenview`](https://github.com/genytri/tokenview) Python CLI as a `/tokens` slash command.

Install the CLI:

```bash
pipx install tokenview
```

Then in any Claude Code session:

```
/tokens budget
/tokens components --top 5
/tokens watch
```

See the main repo README for the full feature list and methodology.
